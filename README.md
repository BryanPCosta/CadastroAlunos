# 📚 Sistema de Cadastro de Alunos - Java Console
Buscando revisar meus conhecimento na lógica de programação criei esse projeto.
**Primeiro projeto do portfólio Java** - Sistema completo de gerenciamento de alunos com menu interativo em console.

## 🎯 Objetivos

- [x] Praticar Orientação a Objetos
- [x] Implementar CRUD completo
- [x] Arquitetura em camadas (entities/services/main)
- [x] Interface console com Scanner
- [x] Busca e validação por matrícula

## 🛠️ Tecnologias
Java 11+
IntelliJ IDEA / VS Code
Git/GitHub

## 📁 Estrutura

src/

├── entities/Aluno.java # Modelo do aluno

├── services/AlunoService.java # Lógica CRUD

└── main/Main.java # Menu interativo

## 🚀 Como rodar

1. Clone: `git clone https://github.com/BryanPCosta/CadastroAlunos`
2. Abra na IDE
3. Execute `Main.java`

## 📱 Demo

=== SISTEMA DE ALUNOS ===

1 - Cadastrar 

2 - Listar 

3 - Buscar

4 - Atualizar 

5 - Remover 

0 - Sair

Exemplo:
   Escolha uma opção: 1
   
    Nome: João Idade: 20 Matrícula: 123 Curso: TI
    ✓ Cadastrado!
